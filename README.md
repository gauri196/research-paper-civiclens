# CivicLens Research Paper

## Title
CivicLens: AI-Powered Civic Awareness Platform

## Abstract
This paper presents CivicLens, a platform that enables citizens to report civic issues, explore government schemes, access candidate information, and interact with an AI-powered civic assistant.
